Broker-Admin-API-Documentation

This Document is used to list all admin APIs of Service Fabrik(SF) broker

Out of Band (OOB) deployment backup APIs

Schedule APIs

Create Schedule

PUT /admin/deployments/:name/schedule_backup

URI Parameters

  • name : string (required) : the deployment name

Body Parameters

  • type : string (required)
    the backup type (either online or offline)
  • repeatInterval : string (required)
    repeat interval of the backup. Supported formats: valid cron expression or human-readable intervals (ex. 4 hours, 6 hours 30 minutes,etc.)
  • timeZone : string (optional)
    timezone to consider when running the batch job. If timezone is not specified, then default timezone of UTC would be considered
  • bosh_director : string (optional)
    bosh director name
